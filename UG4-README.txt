Eigen is provided for UG4 in a separate package in order to conveniantly install
it from ughub. Please have a look at the various licenses under which Eigen
may be used (e.g. under the MPL2 license).

Only necessary header files from Eigen are included in their unmodified form.
If you'd like to use Eigen for your own projects, please consider downloading
a full release from

http://eigen.tuxfamily.org
